# Pattern Memory

每 10 个安全/信息收集任务从最近窗口蒸馏一次 pattern memory。

pattern 关注局部高频模式：

- 哪些打法反复出现
- 哪些信号经常导致转向
- 哪些误报反复浪费时间
- 哪些停止规则应马上固化
