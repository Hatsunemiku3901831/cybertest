# Tactic Memory

每 50 个安全/信息收集任务从多个 pattern 中蒸馏一次 tactic memory。

tactic 关注稳定打法：

- 哪些 pattern 多次有效
- 哪些 pattern 只是偶然成功
- 哪些误报有可量化噪声率
- 哪些规则可以提高为默认优先策略
